# Функциональные тесты

## Модуль: корзина

### Тест-кейс ****№**** 1

**Название:** Отображение пустой корзины

**Описание:** Переход на страницу пустой корзины

**Условие:** Пустая корзина

|  | Шаги | Ожидаемый результат |
| --- | --- | --- |
| 1 | Переход на страницу пустой корзины | Отображение сообщения “Your shopping bag is empty” |

### Тест-кейс ****№ 2****

**Название:** Корректное оформление заказа 

**Описание:** Успешное оформление заказа на странице корзины, содержащей товары

**Условие:** В корзину добавлены товары

|  | Шаги | Ожидаемый результат |
| --- | --- | --- |
| 1 | Переход на страницу корзины | Отображаются данные о товарах: изображение, название, характеристики, стимость. |
| 2 | Нажать кнопку “Proceed to checkout” | Откроется страница с повторным указанием описания продукта, предложением о дополнении корзины, предложением о добавлении сообщения о подарке и предложением о добавлении гравировки. |
| 3 | Нажать кнопку “Proceed to checkout” | Откроется страница с формой заполнения личных данных, адреса доставки и выбором доставки. Также поле с описанием товаров. |
| 4 | Заполнить поля “Title, First name, Last name, Address line 1, Address line 2, Country, City, State, Zip Code, Telephone”  | Значения отобразяться без ошибок, доступна возможность их редактирования |
| 5 | Нажать под товарами кнопку “Procced” | Переход на страницу оплаты |
| 6 | Заполнение данных банковской карты: Card number, Expiry date, CVC/CVV | Информация отобразится в полях формы |
| 7 | Нажать на кнопку “Proceed” | Произойдет переход на страницу подтверждения деталей заказа, где отобразится вся введенная информация и список заказанных товаров.  |
| 8 | Нажать на кнопку “Confirm and order” | Появиться сообщение с текстом “Thanks for the order” |

**Описание:** Переход на страницу пустого wish list’а

Your Wish List is empty.” |

### Тест-кейс ****№ 4****

**Название:** Отображение wish list’а с товарами

**Описание:** Переход на страницу wish list’а после добавления туда товаров

**Условие:**  Пользователь зарегестрирован, в wish list’е ничего нет 

|  | Шаги | Ожидаемый результат |
| --- | --- | --- |

## Модуль: Карточка продукта (информация о продукте)

### Тест-кейс ****№ 5****

**Название:** Корректное отображение информации(текстовой) о продукте, работающие параметры, проверка функциональности навигационной цепочки

**Описание:** Переход на страницу карточки товара

**Условие:**  Наличие товара на сайте

|  | Шаги | Ожидаемый результат |
| --- | --- | --- |
| 1 | Переход на страницу любой карточки товара | Отображаются данные о товаре: изображение, название, характеристики, стимость. |
| 2 | Нажать на кнопку ”Read less” | Текст принимает краткую форму, без лишнего места в контейнере |
| 3 | Нажать на кнопку ”Select size” | Появляется адаптивное меню выбора |
| 4 | Нажать на окна любой дополнительной инфорамции из навигационной цепочки | Корректно открывающееся окно с правильно отображенной инфорацией |

### Тест-кейс ****№ 6****

**Название:** Корректное отображение фотографий; адаптивность

**Описание:** Переход на страницу карточки товара

**Условие:**  Наличие фотографий товара в карточке

|  | Шаги | Ожидаемый результат |
| --- | --- | --- |
| 1 | Переход на страницу любой карточки товара | Просмотр фотографий, подходящих по размеру экрана. Не вылазят за границы экрана, не накладываются на текст |
| 2 | Изменяем размеры до размера планшета | Фотографии изменяют размер и перестраивается расположение |
| 3 | Изменяем размеры экрана до экрана телефона | Фотографии изменяются в размере и появляется слайдер |

## Модуль: Фильтр

### Тест-кейс ****№ 7****

**Название:** Корректное использование фильтра по всем параметрам

**Описание:** Переход на страницу каталога и использование фильтра

**Условие:**  Наличие фильтра

|  | Шаги | Ожидаемый результат |
| --- | --- | --- |
| 1 | Переход на страницу каталога | Наличие фильтра |
| 2 | Выбор любых интересующих характеристик | Показывающиеся карточки сразу же меняются на те, которые соответствуют параметрам, все отображается корректно |

### Тест-кейс ****№ 8****

**Название:** Корректное изменение параметров фильтра (в зависимости от вида поля)

**Описание:** Переход на страницу каталога и использование фильтра

**Условие:**  Наличие фильтра

|  | Шаги | Ожидаемый результат |
| --- | --- | --- |
| 1 | Выбор характеристики в поле фильтра, где возможен только один выбор | Ключ стоит на выбранном варианте, *отображаются товары ему соответствующие |
| 2 | Выбор второй характеристики в поле фильтра, где возможен только один выбор | Ключ меняет расположение, первый очищается |
| 3 | Выбор характеристики в поле фильтра, где возможен множественный выбор | Ключ стоит на выбранном варианте |
| 4 | Выбор второй характеристики в поле фильтра, где возможен множественный выбор | Появляются два ключа, второй остается также выбранным |

## Модуль: Поисковая строка

### Тест-кейс ****№ 9****

**Название:** Поиск товара с помощью поисковой строки

**Описание:** Поиск товара по названию или части названия

**Условие:**  Наличие поисковой строки

|  | Шаги | Ожидаемый результат |
| --- | --- | --- |
| 1 | Нажать на иконку поиска | Отображается поле ввода поисковой строки |
| 2 | Ввод части названия в поисковую строку | Отображение надписи “Do you mean? *name of item*” |
| 3 | Ввод названия в поисковую строку | Отображение запрашиваемого товара |
| 4 | Ввод некорректных данных, либо ввод названия несуществующих товаров | Отображение надписи “No results for this search” |

## Модуль: Чат-ассистент

### Тест-кейс ****№ 10****

**Название:** Использование чат-ассистента

**Описание:** Проверка работы окна чат-ассистента

**Условие:**  Наличие чат-ассистента

|  | Шаги | Ожидаемый результат |
| --- | --- | --- |
| 1 | Нажать на иконку ассистента | Всплывает окно, которое содержит текст “Welcome to Cartier. How may we assist you today?” |
| 2 | Ввод сообщения в строку | Отображение надписи “Thank you for your request” |
| 3 | Нажать на иконку закрытия чата | Отображение кнопок “End Chat” и “Cancel” |
| 4 | Нажать на кнопку “End Chat”  | Окно сворачивается |
| 5 | Нажать на кнопку “Cancel”  | Возвращаемся в чат |
